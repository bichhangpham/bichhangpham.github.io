Program 2: Variable scope, type conversion#

Global variable declaration - declared before main function. All code after this can use the variable

Local variable declaration - created within a function. Only possible to use it within the function

#include<iostream>
using namespace std;

// Declaring a global variable
int total_sum = 0;

// Auto-conversion of type
int sum (double x, double y) {
    total_sum = x + y;
    return total_sum;
}
int main() {
    double a = 2.5;
    double b = 7.0;

    cout << sum(a, b) << endl;
    cout << "The value of total_sum is now: " << total_sum << endl;
}

Program 3: Public attributes#

Attributes which can be accessed directly, even outside of the class definition.

#include <iostream>
using namespace std;

class Box {
    public:
        double length;
        double breadth;
        double height;
};

double getVolume(Box box) {
    return box.length * box.breadth * box.height;
}

int main() {
    Box myBox;

    myBox.length = 4;
    myBox.breadth = 9;
    myBox.height = 5;

    cout << "The volume for my box is: " << getVolume(myBox) << endl;

}

Program 4: Private attributes#

Variables are declared as private.

Attributes which cannot be accessed outside of the class, and must be done through the methods within this class

Constructor is initialising the attributes.

Recommended to define classes this way rather than using public attributes as outlined in the previous program.

#include <iostream>
using namespace std;

class Box {
    private:
        double length;
        double breadth;
        double height;
    public:
        Box(double l, double b, double h) {
            length = l;
            breadth = b;
            height = h;

        }
        double getVolume() {
            return length * breadth * height;
        }
};


int main() {
    Box myBox(9, 5, 7);

    cout << "The volume for my box is: " << myBox.getVolume() << endl;

}

Program 6: Pointer#

Stores the memory address of a variable or object. Things to note:

  • *ptr will return the value that is stored in the memory address that pointer is pointing to
  • &ptr will return the memory location the pointer is pointing to
#include <iostream>
using namespace std;

class Box {
    private:
        double length;
        double breadth;
        double height;
    public:
        Box(double l, double b, double h) {
            length = l;
            breadth = b;
            height = h;

        }
        double getVolume() {
            return length * breadth * height;
        }
        void printVolume() {
            cout << "Volume of box: " << getVolume() << endl;
        }
};


int main() {
    // Calling the object's method directly
    Box myBox(9, 5, 7);
    myBox.printVolume();
    
    // Using a pointer to the class object
    // Method is called using a pointer
    // Pointer is then deleted after 
    Box * ptr = new Box(9, 5, 7);
    ptr -> printVolume();
    delete ptr;

}
  • Created the class object the new keyword will dynamically allocate memory. It will return a memory address to the newly created object.
  • The class object has no name, but its memory space is allocated, which is then assigned to the pointer.
  • Using â†’ will allow accessing attributes and methods
  • We also need to ensure we delete the ptr to ensure memory space is freed up

Program 7: Dereferencing a pointer#

In the previous program, we called the method using ptr -> printVolume();

In this example, we de-reference a pointer, and we are calling a method using the class object that was de-referenced by that pointer.

#include <iostream>
using namespace std;

class Box {
    private:
        double length;
        double breadth;
        double height;
    public:
        Box(double l, double b, double h) {
            length = l;
            breadth = b;
            height = h;

        }
        double getVolume() {
            return length * breadth * height;
        }
        void printVolume() {
            cout << "Volume of box: " << getVolume() << endl;
        }
};


int main() {

    Box * ptr = new Box(9, 5, 7);
    (*ptr).printVolume();
    delete ptr;

}

calling method using ptr:

  • ptr -> printVolume(); calling method after ptr has been deferenced:
  • (*ptr).printVolume();

Program 8: Pointer to simple data types#

Create a pointer to point to a simple data type (double)

#include <iostream>
using namespace std;

int main() {
    // Initialise pointer with null
    int * intPtr = nullptr;
    // int * intPtr = NULL;
    // Allocate memory space to ptr
    intPtr = new int;
    // Store value in allocated address
    * intPtr = 2000; 
    // Prints value 
    cout << *intPtr + 1000 << endl;
    //Remove pointer
    delete intPtr;
}

Program 9: Reference#

Normally not used, but used when passing parameters so that original object/variable is also modified

#include <iostream>

using namespace std;

int add_length(int& x, int &y) {
    x = x + 10;
    return x + y;
}

int main() {
    int x = 10;
    int y = 10;

    cout << "Adding x and y = " << x + y << endl;
    cout << "Adding using add_length function = " << add_length(x, y) << endl;
    cout << "Adding x and y again = " << x + y << endl;
}

Result of the above code:

Adding x and y = 20
Adding using add_length function = 30
Adding x and y again = 30

Program 10: Without using a reference#

The example below is the same as program 10, except the & value is not used in the arguments for the add_length function.

The original memory address and value is not modified.

#include <iostream>

using namespace std;
//Removing the ampersand
int add_length(int x, int y) {
    x = x + 10;
    return x + y;

    cout << "Adding using add_length function = " << add_length(x, y) << endl;

The result should be:

Adding x and y = 20
Adding using add_length function = 30
Adding x and y again = 20
